# Sakila SQL Data Analysis

SQL data analysis project using the **Sakila sample database** to explore customer behavior, rental activity, film performance and business insights.

This project was developed as part of the **Adalab Data Analytics & AI Bootcamp** and has been adapted for portfolio purposes.

---

## Overview

The goal of this project was to practice SQL querying and relational database analysis using the Sakila sample database.

Sakila represents a fictional video rental business and includes information about customers, rentals, films, actors, categories, payments, staff and stores.

The project focuses on extracting insights from a relational database through SQL queries, joins, aggregations, subqueries and business-oriented analysis.

---

## Objective

The main objective was to use SQL to answer business questions related to:

- Customer behavior
- Film rental activity
- Store performance
- Payment and revenue patterns
- Film categories
- Inventory and rental trends
- Actor and film relationships

---

## Dataset Context

The project is based on the **Sakila sample database**, a relational database commonly used to practice SQL and MySQL.

The database includes multiple related tables, such as:

- `customer`
- `rental`
- `payment`
- `film`
- `actor`
- `category`
- `inventory`
- `store`
- `staff`
- `address`
- `city`
- `country`

These tables allow the analysis of a fictional video rental business from different perspectives.
